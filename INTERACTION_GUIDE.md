# 🌍 交互操作指南 | Interaction Guide

## 📖 如何查看旅行日记？

### 方法 1：点击底部城市名称 ⭐️ **推荐**

- 在底部控制栏中，**直接点击城市名称**
- 城市名称有悬浮高亮效果，并带有 📖 图标提示
- 这是最简单直观的方式！

### 方法 2：双击右侧时间轴

- 在右侧时间轴列表中，**单击切换城市，双击打开日记**
- 当前选中的城市会有高亮背景
- 鼠标悬停时会显示提示："点击切换，双击查看日记"

### 方法 3：点击地球上的标记点

- 在 3D 地球上，找到城市的**发光标记点**
- 点击标记点即可打开该城市的日记
- 相同位置的多个访问会显示选择列表

## 🎮 其他交互功能

### 🔍 搜索功能

- **点击顶部搜索框**，输入关键词（城市名、心情标签等）
- 支持中英文搜索
- 搜索时显示清除按钮 (✕)

### ⏮️ ⏭️ 时间轴导航

- 使用底部控制栏的 **← →** 按钮切换城市
- 也可以直接点击右侧时间轴的城市名称
- 支持自动播放功能 (▶️ 按钮)

### 🌓 主题切换

- 点击顶部 **☀️/🌙** 图标切换日夜主题
- 暗色模式：夜空星辰 + 深色背景
- 亮色模式：白天地球 + 天空蓝背景

### 📊 统计面板

- 点击顶部 **📊** 图标查看旅行统计
- 显示：总旅行次数、城市数量、国家数量、总距离

### ▶️ 自动播放

- 点击顶部 **▶️** 图标启动自动播放
- 每 2 秒自动切换到下一个城市
- 播放到最后会自动停止

## 💡 使用提示

### 首次访问

- 首次打开页面会显示**使用提示浮窗**
- 介绍三种打开日记的方法
- 点击"知道了"关闭提示

### 视觉反馈

- ✨ 所有可点击元素都有**悬浮高亮**效果
- 🎯 当前选中的城市有**金色/蓝色**高亮
- 📖 底部城市名称右上角有书本图标闪烁提示

### 键盘快捷键（即将支持）

- `←` / `→` : 切换上一个/下一个城市
- `Space` : 打开当前城市日记
- `Esc` : 关闭日记弹窗

## 🚀 高级功能

### 添加照片

在 `TRAVEL_DATA` 中为任意城市添加 `photos` 字段：

```javascript
{
  id: 1,
  photos: [
    "https://example.com/photo1.jpg",
    "https://example.com/photo2.jpg"
  ],
  // ... 其他字段
}
```

### 自定义颜色

每个城市的 `moodColor` 字段定义了：

- 地球上标记点的颜色
- 连接弧线的颜色
- 脉冲光环的颜色

### 搜索标签

通过 `aiTags` 字段添加可搜索的标签：

```javascript
aiTags: ["Beach", "Summer", "Relax"];
```
